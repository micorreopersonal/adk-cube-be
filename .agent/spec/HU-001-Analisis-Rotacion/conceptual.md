# HU-001: Análisis de Rotación Mensual (Conceptual)

## 1. Visión del Negocio
El área de Gestión de Talento necesita una herramienta que automatice la consolidación de métricas de rotación al cierre de cada mes. Actualmente, este proceso es manual y propenso a errores, lo que retrasa la toma de decisiones estratégicas para la retención.

## 2. Objetivos
*   Reducir el tiempo de generación del reporte de rotación de días a segundos.
*   Garantizar la integridad de los datos eliminando cálculos manuales ("Regla de Oro: Cero Alucinaciones").
*   Proporcionar insights narrativos que expliquen las variaciones de los indicadores.

## 3. Valor para el Usuario
El HR Manager podrá consultar desde un chat seguro el estado de la rotación, compararlo con el promedio anual y obtener alertas sobre la fuga de talento crítico (Hipers/Hipos) sin depender de analistas de datos.
